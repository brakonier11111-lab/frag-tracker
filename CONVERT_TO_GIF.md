# Инструкция по конвертации видео в GIF с высоким FPS

## Увеличение FPS в GIF

### Вариант 1: Онлайн конвертеры с настройкой FPS
1. **EZGIF** - https://ezgif.com/video-to-gif
   - Загрузите ваш MOV файл
   - Установите FPS: **15-20** (вместо стандартных 10)
   - Размер: 300x420 пикселей
   - Оптимизация: включена

2. **Convertio** - https://convertio.co/mov-gif/
   - Выберите "Advanced settings"
   - FPS: **20-25**
   - Размер: 300x420

3. **Online-Convert** - https://www.online-convert.com/ru/convert-to-gif
   - Настройки качества: Высокое
   - FPS: **15-20**

### Вариант 2: FFmpeg (командная строка) - РЕКОМЕНДУЕТСЯ
```bash
# Установите FFmpeg, затем выполните:
ffmpeg -i "Widget Peregrev.mov" -vf "fps=20,scale=300:-1:flags=lanczos" -c:v gif "Widget Peregrev_20fps.gif"

# Или с еще более высоким FPS:
ffmpeg -i "Widget Peregrev.mov" -vf "fps=25,scale=300:-1:flags=lanczos" -c:v gif "Widget Peregrev_25fps.gif"

# Для максимального качества (больший размер файла):
ffmpeg -i "Widget Peregrev.mov" -vf "fps=30,scale=300:-1:flags=lanczos" -c:v gif "Widget Peregrev_30fps.gif"
```

### Вариант 3: VLC Media Player
1. Откройте VLC
2. Медиа → Конвертировать/Сохранить
3. Добавьте ваш MOV файл
4. Выберите профиль "Video for YouTube HD"
5. Нажмите "Настроить профиль"
6. Вкладка "Видео" → Частота кадров: **20-25**
7. Измените расширение на .gif

### Вариант 4: Adobe Photoshop
1. Откройте MOV файл в Photoshop
2. Окно → Шкала времени
3. Настройки → Частота кадров: **20-25 FPS**
4. Файл → Экспортировать → Сохранить для Web (устаревший)
5. Выберите GIF, оптимизация включена

## Рекомендуемые настройки для высокого FPS:
- **FPS: 20-25** (оптимальный баланс качества и размера)
- **Размер: 300x420 пикселей** (как в виджете)
- **Цвета: 256 цветов максимум**
- **Оптимизация: включена**
- **Dithering: включен** (для плавности)

## Альтернативные форматы с высоким FPS:
- **WebM** - лучшая совместимость, поддерживает высокий FPS
- **MP4** - универсальный формат, хорошее качество
- **APNG** - анимированный PNG, высокое качество

## После конвертации:
1. Поместите новый GIF в папку `assets/`
2. Замените `2025-06-24-11-42-45.gif` на новый файл
3. Или обновите путь в виджете

## Поддержка альфа-канала (прозрачности)

### WebM с альфа-каналом:
```bash
# Конвертация в WebM с альфа-каналом:
ffmpeg -i "Widget Peregrev.mov" -c:v libvpx-vp9 -pix_fmt yuva420p -crf 30 -b:v 0 -b:a 128k -c:a libopus "Widget Peregrev_alpha.webm"

# Или с более высоким качеством:
ffmpeg -i "Widget Peregrev.mov" -c:v libvpx-vp9 -pix_fmt yuva420p -crf 20 -b:v 0 -b:a 128k -c:a libopus "Widget Peregrev_alpha_hq.webm"
```

### APNG с альфа-каналом:
```bash
# Конвертация в APNG с альфа-каналом:
ffmpeg -i "Widget Peregrev.mov" -vf "fps=20,scale=300:-1:flags=lanczos" -plays 0 "Widget Peregrev_alpha.png"
```

### Важные параметры для альфа-канала:
- **`-pix_fmt yuva420p`** - формат пикселей с альфа-каналом
- **`-plays 0`** - бесконечное воспроизведение для APNG
- **Прозрачный фон** - убедитесь, что исходное видео имеет прозрачность

## Проверка результата:
- Откройте виджет в браузере
- Проверьте плавность анимации
- Если слишком быстро - уменьшите FPS
- Если слишком медленно - увеличьте FPS
