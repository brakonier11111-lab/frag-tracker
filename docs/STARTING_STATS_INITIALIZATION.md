# 🚀 Инициализация стартовых значений статистики

## 🎯 Описание
Система для инициализации стартовых значений статистики боев и фрагов на основе реальных данных игрока.

## 📊 Стартовые значения
- **Всего боев**: 19
- **Всего фрагов**: 37
- **Боев без фрагов**: 4
- **Боев с фрагами**: 15

## 🔧 Расчеты

### Распределение фрагов
```
Боев с фрагами: 19 - 4 = 15
Фрагов за бой: 37 ÷ 15 = 2 (целая часть)
Остаток фрагов: 37 - (2 × 15) = 7
```

### Распределение по боям
```
• 15 боев с фрагами:
  - 1-й бой: 2 + 7 = 9 фрагов
  - 14 боев: по 2 фрага

• 4 боя: по 0 фрагов
```

## 🚀 Способы инициализации

### 1. Через админ-панель
1. Откройте http://localhost:3000/admin
2. Нажмите "🚀 ИНИЦИАЛИЗАЦИЯ СТАРТОВЫХ ЗНАЧЕНИЙ"
3. Введите значения:
   - Всего боев: `19`
   - Всего фрагов: `37`
   - Боев без фрагов: `4`
4. Подтвердите инициализацию

### 2. Через скрипт
```bash
# Запуск скрипта инициализации
initialize-starting-stats.bat
```

### 3. Через API
```bash
curl -X POST http://localhost:3000/api/initialize-starting-stats \
  -H "Content-Type: application/json" \
  -d '{
    "totalBattles": 19,
    "totalFrags": 37,
    "battlesWithoutFrags": 4
  }'
```

## 📋 Что происходит при инициализации

### 1. Очистка данных
- Удаляются все записи из таблицы `frag_stats`
- Сбрасывается статистика фрагов

### 2. Создание боев
- Создается 19 записей боев
- Каждый бой получает уникальное время
- Фраги распределяются согласно расчетам

### 3. Обновление состояния
- Обновляется `lesta_last_battles = 19`
- Обновляется `lesta_last_frags = 37`
- Обновляется `lesta_previous_frags = 37`
