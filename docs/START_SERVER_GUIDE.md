# Единый бат-файл для запуска Frag-tracker Server

## Описание
Создан единый бат-файл `start-server-universal.bat`, который автоматически выполняет все необходимые шаги для запуска сервера:

1. **Проверка Node.js** - проверяет наличие Node.js
2. **Проверка package.json** - проверяет наличие файла проекта
3. **Остановка процессов** - останавливает существующие процессы Node.js
4. **Установка зависимостей** - выполняет `npm install`
5. **Инициализация БД** - запускает `init-database.js`
6. **Запуск сервера** - запускает `node server.js`

## Использование

### Способ 1: Двойной клик
Просто дважды кликните на файл `start-server-universal.bat` в проводнике Windows.

### Способ 2: Командная строка
```cmd
start-server-universal.bat
```

### Способ 3: PowerShell
```powershell
.\start-server-universal.bat
```

## Что делает бат-файл

1. **Проверяет окружение**:
   - Наличие Node.js
   - Наличие package.json
   - Останавливает старые процессы

2. **Устанавливает зависимости**:
   - Выполняет `npm install`
   - Проверяет успешность установки

3. **Инициализирует базу данных**:
   - Запускает `init-database.js`
   - Создает все необходимые таблицы
   - Вставляет начальные данные

4. **Запускает сервер**:
   - Запускает `node server.js`
   - Показывает доступные URL
   - Ожидает остановки сервера

## Доступные страницы после запуска

- **Главная**: http://localhost:3000
- **Админ панель**: http://localhost:3000/admin.html
- **Аналитика**: http://localhost:3000/donations-analytics.html
- **Проверка БД**: http://localhost:3000/database-init-check.html
- **Тест донатов**: http://localhost:3000/test-donations.html

## Остановка сервера

Для остановки сервера нажмите `Ctrl+C` в окне командной строки.

## Требования

- **Node.js** версии 14 или выше
- **npm** (устанавливается вместе с Node.js)
- **Windows** (для бат-файла)

## Устранение проблем

### Ошибка "Node.js не найден"
- Установите Node.js с https://nodejs.org/
- Перезапустите командную строку

### Ошибка "package.json не найден"
- Убедитесь, что вы находитесь в папке проекта
- Проверьте наличие файла package.json

### Ошибка установки зависимостей
- Запустите бат-файл от имени администратора
- Проверьте подключение к интернету

### Ошибка инициализации БД
- Проверьте наличие файла init-database.js
- Убедитесь, что у вас есть права на запись в папку

## Альтернативные файлы

## Логи

Все операции логируются в консоль с индикаторами:
- `OK:` - успешное выполнение
- `ОШИБКА:` - ошибка выполнения
- `INFO:` - информационное сообщение
