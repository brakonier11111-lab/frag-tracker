## 🎯 Новая система

### Основные возможности

1. **Управление сбором**
   - Настройка названия сбора
   - Установка целевой суммы
   - Ручное управление текущей суммой
   - Добавление ручных донатов

2. **Превью в реальном времени**
   - Визуальное отображение прогресса
   - Автоматическое обновление
   - Статистика донатов

3. **История донатов**
   - Последние 20 донатов
   - Информация о донатере и сумме
   - Временные метки

4. **Виджет для OBS**
   - Прозрачный фон
   - Красивая анимация прогресса
   - Автоматическое обновление через WebSocket

## 🚀 Использование

### Страница управления
Откройте: `http://localhost:3000/donation-goal.html`

### Виджет для OBS
URL: `http://localhost:3000/widget/donation-goal`

**Настройка в OBS:**
1. Добавьте Browser Source
2. URL: `http://localhost:3000/widget/donation-goal`
3. Ширина: 800px
4. Высота: 200px
5. Включите "Обновить кэш браузера"

## 📡 API Endpoints

### Получить состояние
```
GET /api/donation-goal
или
GET /api/donation-bar/state
```

### Обновить состояние
```
PUT /api/donation-goal
Content-Type: application/json

{
  "title": "Название сбора",
  "targetAmount": 10000,
  "currentAmount": 0
}
```

### Добавить ручной донат
```
POST /api/donation-goal/manual-donation
Content-Type: application/json

{
  "amount": 100
}
```

### Обнулить сбор
```
POST /api/donation-goal/reset
```

### Получить историю
```
GET /api/donation-goal/history
```
