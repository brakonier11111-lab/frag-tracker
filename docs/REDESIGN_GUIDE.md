# 🎨 Руководство по новому дизайну приложения

## Что было создано

### ✅ Новая структура приложения

1. **Боковое меню (Sidebar)** - `/public/components/sidebar.html`
   - Группировка по разделам (Трекеры, Донаты, Статистика и т.д.)
   - Сворачивание/разворачивание
   - Автоматическое выделение активной страницы
   - Адаптивный дизайн для мобильных

2. **Верхняя панель (Header)** - `/public/components/header.html`
   - Хлебные крошки (навигация)
   - Статистика в реальном времени
   - Кнопки действий
   - Адаптивное мобильное меню

3. **Система макета** - `/public/styles/layout.css`
   - Фиксированное боковое меню
   - Фиксированная верхняя панель
   - Адаптивный основной контент
   - Плавные переходы

4. **Обновленная главная страница** - `/public/index-new.html`
   - Использует новую структуру
   - Современный дизайн
   - Статистика на главной

## 🚀 Как использовать

### Вариант 1: Использовать новую главную страницу

Переименуйте файлы:
```bash
# Сделайте резервную копию старой
mv public/index.html public/index-old.html

# Используйте новую
mv public/index-new.html public/index.html
```

### Вариант 2: Обновить существующую страницу

Добавьте в начало `<body>` вашей страницы:

```html
<div class="app-layout">
    <!-- Боковое меню -->
    <div id="sidebar-placeholder"></div>
    
    <!-- Верхняя панель -->
    <div id="header-placeholder"></div>
    
    <!-- Основной контент -->
    <main class="app-main">
        <!-- Ваш контент здесь -->
    </main>
</div>

<!-- Подключите layout.css -->
<link rel="stylesheet" href="/styles/layout.css">

<!-- Скрипт загрузки компонентов -->
<script>
    Promise.all([
        fetch('/components/sidebar.html').then(r => r.text()),
        fetch('/components/header.html').then(r => r.text())
    ]).then(([sidebar, header]) => {
        document.getElementById('sidebar-placeholder').innerHTML = sidebar;
        document.getElementById('header-placeholder').innerHTML = header;
    });
</script>
```

## 🎨 Особенности дизайна

### Боковое меню
- **Ширина**: 280px (развернуто), 80px (свернуто)
- **Группировка**: Разделы с заголовками
- **Активное состояние**: Подсветка и индикатор
- **Адаптивность**: Скрывается на мобильных, открывается по кнопке

### Верхняя панель
- **Высота**: 70px
- **Хлебные крошки**: Показывают текущую страницу
- **Статистика**: Обновляется каждые 30 секунд
- **Мобильное меню**: Кнопка для открытия сайдбара

### Основной контент
- **Отступы**: Автоматически учитывают меню
- **Адаптивность**: Полная ширина на мобильных

## 🔧 Кастомизация

### Изменение ширины сайдбара

В `layout.css`:
```css
.sidebar {
    width: 280px; /* Измените на нужное значение */
}

.sidebar.collapsed {
    width: 80px; /* Ширина свернутого состояния */
}
```

### Добавление новых пунктов меню

В `sidebar.html` добавьте в нужную секцию:
```html
<a href="/your-page.html" class="nav-item" data-page="your-page">
    <span class="nav-icon">🎯</span>
    <span class="nav-text">Ваш пункт</span>
</a>
```

### Изменение цветов

Используйте CSS переменные из `design-system.css`:
```css
.nav-item.active {
    background: rgba(179, 0, 255, 0.15); /* Измените цвет */
    color: var(--color-neon-purple);
}
```

## 📱 Адаптивность

- **Desktop** (>768px): Полное боковое меню, все функции
- **Tablet** (≤768px): Скрытое меню, открывается по кнопке
- **Mobile** (≤768px): Мобильное меню, компактная статистика

## 🐛 Решение проблем

### Меню не загружается
- Проверьте, что сервер отдает `/components/sidebar.html`
- Проверьте консоль браузера на ошибки

### Контент перекрывается меню
- Убедитесь, что подключен `layout.css`
- Проверьте, что используется класс `app-main`

### Меню не сворачивается
- Проверьте, что скрипт загружен
- Проверьте localStorage в браузере
