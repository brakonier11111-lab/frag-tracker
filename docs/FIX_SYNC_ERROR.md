# 🔧 Исправление ошибки syncLestaGamesStats

## 🐛 Проблема
Ошибка: `ReferenceError: syncLestaGamesStats is not defined`

Эта ошибка возникала в функции `autoRefreshFragStats()` при попытке вызвать несуществующую функцию `syncLestaGamesStats`.

## ✅ Решение

### Исправление в server.js
Заменили вызов несуществующей функции на корректную логику:

```javascript
// Было (неправильно):
if (state.lesta_account_id && state.lesta_application_id) {
    syncLestaGamesStats(state.lesta_account_id, state.lesta_application_id);
}

// Стало (правильно):
if (state.lesta_account_id && state.lesta_application_id) {
    // Запускаем синхронизацию Lesta Games
    const stats = await getLestaPlayerStats();
    if (stats) {
        console.log('✅ Синхронизация Lesta Games выполнена');
    }
}
```

## 🔍 Причина ошибки
В коде автообновления была попытка вызвать функцию `syncLestaGamesStats`, которая не была определена в проекте. Вместо неё нужно использовать существующую функцию `getLestaPlayerStats()`.
