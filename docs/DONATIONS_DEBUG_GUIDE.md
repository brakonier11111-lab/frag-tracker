# 🔍 Диагностика проблемы с донатами

## 📊 Симптомы
- Статус системы показывает "Работает" для всех компонентов
- Донаты не отображаются в интерфейсе
- Донаты не считаются в статистике

## 🛠️ Инструменты диагностики

### 1. Диагностическая кнопка в админ-панели
1. Откройте http://localhost:3000/admin
2. Нажмите кнопку "🔍 ДИАГНОСТИКА ДОНАТОВ"
3. Изучите результаты в модальном окне

### 2. API эндпоинты для проверки
- `GET /api/debug-donations` - полная диагностика
- `GET /api/da-api-test` - тест API DonationAlerts
- `POST /api/force-check-donations` - принудительная проверка

### 3. Диагностическая страница
- http://localhost:3000/donation-alerts-test.html

## 🔍 Возможные причины проблемы

### 1. Проблема с токеном
**Симптомы:**
- API тест показывает ошибку 401
- В диагностике "Токен: НЕТ"

**Решение:**
1. Выполните OAuth авторизацию заново
2. Проверьте срок действия токена
3. Обновите токен в БД

### 2. Проблема с API
**Симптомы:**
- API тест показывает ошибку
- Нет донатов в ответе API

**Решение:**
1. Проверьте есть ли донаты в личном кабинете DonationAlerts
2. Отправьте тестовый донат
3. Проверьте права доступа приложения

### 3. Проблема с сохранением в БД
**Симптомы:**
- API возвращает донаты, но в БД их нет
- Ошибки в логах сервера

**Решение:**
1. Проверьте логи сервера
2. Проверьте права доступа к БД
3. Перезапустите сервер

### 4. Проблема с опросом
**Симптомы:**
- Интервал опроса неактивен
- Опрос не запускается

**Решение:**
1. Перезапустите сервер
2. Проверьте настройки опроса
3. Проверьте логи сервера

## 🚀 Пошаговое решение

### Шаг 1: Запуск диагностики
1. Откройте админ-панель
2. Нажмите "🔍 ДИАГНОСТИКА ДОНАТОВ"
3. Изучите результаты

### Шаг 2: Проверка токена
Если токен отсутствует:
1. Нажмите "🔑 НАСТРОИТЬ DA"
2. Авторизуйтесь в DonationAlerts
3. Разрешите доступ приложению

### Шаг 3: Проверка донатов
Если токен есть, но донатов нет:
1. Отправьте тестовый донат в DonationAlerts
2. Нажмите "🔄 ПРОВЕРИТЬ ДОНАТЫ"
3. Проверьте результаты

### Шаг 4: Проверка API
Если донаты есть в DonationAlerts, но не приходят:
1. Проверьте права доступа приложения
2. Обновите токен
3. Проверьте логи сервера

### Шаг 5: Проверка сохранения
Если API возвращает донаты, но они не сохраняются:
1. Проверьте логи сервера на ошибки
2. Проверьте права доступа к БД
3. Перезапустите сервер

## 📋 Чек-лист диагностики

- [ ] DonationAlerts API настроен

- [ ] Есть донаты в личном кабинете DonationAlerts

## 🔧 Дополнительные команды

### Проверка логов сервера
Следите за логами сервера на предмет:
- `✅ DonationAlerts OAuth успешно!`
- `🔍 Запрос донатов с токеном...`
- `✅ Получены донаты: X`
- `❌ Ошибка API DonationAlerts`

### Проверка БД через SQLite
```sql
-- Проверка токена
SELECT da_access_token FROM app_state WHERE id = 1;

-- Проверка донатов
SELECT COUNT(*) FROM donations;
SELECT * FROM donations ORDER BY created_at DESC LIMIT 10;
```

### Тест API вручную
```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
     https://www.donationalerts.com/api/v1/alerts/donations
```
