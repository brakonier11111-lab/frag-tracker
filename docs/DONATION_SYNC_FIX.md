# 🔧 Решение проблемы с отображением суммы донатов

## 🚨 **Проблема:**
- В режиме фрагов отображается: "Всего донатов 2666₽"
- В виджете показывается старая сумма
- Кнопки изменения баланса не отображаются

## ✅ **Решение:**

### 1. **Проверка сервера:**
- Сервер должен быть запущен из правильной директории
- Используйте `start_server.bat` для запуска
- Проверьте, что порт 3000 свободен

### 2. **Тестовая страница:**
- Откройте: `http://localhost:3000/test-donation-controls.html`
- Проверьте работу всех API endpoints
- Протестируйте функции управления донатами

### 3. **Проверка кнопок в режиме фрагов:**
- Откройте: `http://localhost:3000/mode1-frag-tracker.html`
- Найдите карточку "Всего донатов"
- Должны быть видны кнопки:
  - 🔄 **Обнулить** (красная)
  - 💰 **Стартовая сумма** (желтая)

### 4. **Синхронизация виджетов:**
- Основной виджет: `http://localhost:3000/widget-mode2.html`
- Копия виджета: `http://localhost:3000/widget-mode2-copy.html`
- Оба должны показывать актуальную сумму

## 🔍 **Диагностика:**

### **API Endpoints:**
- `GET /api/donations-stats` - получить статистику
- `POST /api/reset-donations-count` - обнулить все донаты
- `POST /api/fix-total-donated` - исправить рассинхронизацию

### **Функции:**
- `resetDonationTotal()` - обнуляет сумму и количество
- `setStartingAmount()` - устанавливает стартовую сумму
- `loadDonationStats()` - загружает актуальную статистику

## 🎯 **Пошаговое решение:**

### **Шаг 1: Запуск сервера**
```bash
# Остановить все процессы Node.js
taskkill /F /IM node.exe

# Запустить сервер
start_server.bat
```

### **Шаг 2: Проверка API**
1. Откройте `http://localhost:3000/test-donation-controls.html`
2. Нажмите "🔄 Обновить статистику"
3. Проверьте, что данные загружаются

### **Шаг 3: Тест функций**
1. Нажмите "🔄 Обнулить сумму и количество"
2. Подтвердите действие
3. Проверьте, что сумма обнулилась

### **Шаг 4: Проверка виджетов**
1. Откройте режим фрагов
2. Найдите кнопки под суммой донатов
3. Проверьте, что они работают

## 🚀 **Ожидаемый результат:**

### **В режиме фрагов:**
- Отображается: "Всего донатов: 0₽" (или актуальная сумма)
- Видны кнопки "🔄 Обнулить" и "💰 Стартовая сумма"
- Кнопки работают и обновляют сумму

### **В виджетах:**
- Отображается актуальная сумма из базы данных
- Сумма синхронизируется между всеми виджетами
- Обновления происходят в реальном времени

## 🔧 **Если проблема остается:**

### **Проверьте:**
1. **Сервер запущен** - порт 3000 должен быть занят
2. **База данных** - файл `frag_tracker.db` должен существовать
3. **API работает** - тестовая страница должна загружать данные
4. **Кнопки видны** - CSS стили должны применяться

### **Перезапуск:**
1. Остановите сервер: `Ctrl+C` в терминале
2. Запустите заново: `start_server.bat`
3. Обновите страницы в браузере

Теперь все должно работать корректно! 🎉
