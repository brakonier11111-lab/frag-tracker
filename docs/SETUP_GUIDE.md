# 🚀 Руководство по настройке интеграций

## 📊 Текущий статус системы

**Это нормально, если вы видите:**
- DonationAlerts: Не подключен
- DonatePay: Не настроен  
- Lesta Games: Не авторизован
- WebSocket: Отключен

Это означает, что интеграции еще не настроены.

## 🔗 Настройка DonationAlerts

### Шаг 1: OAuth авторизация
1. Откройте http://localhost:3000/admin
2. В разделе "🔗 DONATIONALERTS" нажмите **"🔑 НАСТРОИТЬ DA"**
3. В открывшемся окне авторизуйтесь в DonationAlerts
4. Разрешите доступ приложению
5. После успешной авторизации статус изменится на "Подключен"

### Шаг 2: Проверка настройки
После авторизации вы должны увидеть:
- ✅ DonationAlerts: Подключен
- ✅ WebSocket: Подключен (если есть активные соединения)

### Шаг 3: Тестирование
1. Нажмите **"🧪 ТЕСТ DA"** для проверки API
2. Нажмите **"🔄 ПРОВЕРИТЬ ДОНАТЫ"** для получения донатов
3. Отправьте тестовый донат в DonationAlerts

## 💰 Настройка DonatePay (опционально)

### Если хотите использовать DonatePay:
1. Получите API ключ в личном кабинете DonatePay
2. Нажмите **"🔗 НАСТРОИТЬ DONATEPAY"**
3. Введите API ключ
4. Настройте webhook URL: `http://localhost:3000/webhook/donatepay`

## 🎮 Настройка Lesta Games (опционально)

### Если хотите использовать статистику World of Tanks Blitz:
1. Получите Application ID в Lesta Games Developer Portal
2. Нажмите **"🎮 НАСТРОИТЬ LESTA GAMES"**
3. Выполните авторизацию
4. Настройте синхронизацию статистики

## 🔍 Диагностика проблем

### Если DonationAlerts не подключается:
1. Проверьте, что Client ID и Client Secret настроены
2. Убедитесь, что Redirect URI правильный: `http://localhost:3000/auth/callback`
3. Проверьте, что приложение активно в DonationAlerts
4. Выполните авторизацию заново

### Если донаты не отображаются:
1. Нажмите **"🔍 ДИАГНОСТИКА ДОНАТОВ"**
2. Изучите результаты диагностики
3. Отправьте тестовый донат в DonationAlerts
4. Проверьте логи сервера

## 🆘 Решение проблем

### Проблема: "Не удается подключиться к DonationAlerts"
**Решение:**
1. Проверьте интернет-соединение
2. Проверьте статус DonationAlerts: https://status.donationalerts.com/
3. Попробуйте авторизацию заново
4. Проверьте настройки файрвола

### Проблема: "OAuth авторизация не работает"
**Решение:**
1. Проверьте Client ID и Client Secret
2. Убедитесь, что Redirect URI правильный
3. Проверьте, что приложение активно
4. Очистите кэш браузера

### Проблема: "Донаты не отображаются"
**Решение:**
1. Выполните диагностику донатов
2. Проверьте, есть ли донаты в личном кабинете
3. Отправьте тестовый донат
4. Проверьте логи сервера
