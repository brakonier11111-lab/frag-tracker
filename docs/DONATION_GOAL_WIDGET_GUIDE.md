# Виджет сбора донатов

Современный виджет для сбора донатов с красивой анимированной полоской прогресса и полным управлением.

## Возможности

### 🎨 Современный дизайн
- Стеклянный эффект с размытием фона
- Градиентные цвета и анимации
- Адаптивный дизайн для разных размеров экрана
- Красивые эффекты заполнения полоски прогресса

### 📊 Полная статистика
- Текущая сумма сбора
- Целевая сумма
- Процент выполнения
- Количество донатов
- Средний чек
- Время последнего доната

### ⏰ Таймер обратного отсчета
- Установка срока окончания сбора
- Отображение дней, часов, минут до окончания
- Автоматическое скрытие при отсутствии срока

### 🎛️ Панель управления
- Изменение названия и описания сбора
- Установка целевой суммы
- Ручное изменение текущей суммы
- Добавление ручных донатов
- Установка срока окончания
- Обнуление сбора
- Экспорт данных

### 🔗 Интеграция с реальными донатами
- Автоматическое обновление при получении донатов из DonationAlerts
- Автоматическое обновление при получении донатов из DonatePay
- История всех донатов (ручных и автоматических)

## Использование

### 1. Основная страница управления
Откройте `http://localhost:3000/donation-goal-widget.html` для настройки и управления сбором.

### 2. Виджет для OBS
Используйте `http://localhost:3000/widget/donation-goal` в OBS как Browser Source.

### 3. Инициализация базы данных
Перед первым использованием запустите:
```bash
node init-db.js
```
или используйте bat файл:
```bash
init-donation-goal-db.bat
```

## API Endpoints

### Получить данные цели
```
GET /api/donation-goal
```

### Обновить настройки цели
```
PUT /api/donation-goal
Content-Type: application/json

{
  "title": "Название сбора",
  "description": "Описание сбора",
  "targetAmount": 10000,
  "currentAmount": 0,
  "endDate": "2024-12-31T23:59:59.000Z"
}
```

### Добавить ручной донат
```
POST /api/donation-goal/manual-donation
Content-Type: application/json

{
  "amount": 100
}
```

### Обнулить сбор
```
POST /api/donation-goal/reset
```

### Получить историю донатов
```
GET /api/donation-goal/history
```

## Настройка в OBS

1. Добавьте новый Browser Source
2. URL: `http://localhost:3000/widget/donation-goal`
3. Ширина: 800px (или по вашему усмотрению)
4. Высота: 200px (или по вашему усмотрению)
5. Обновите кэш браузера: включено

## Особенности

- **Автообновление**: Виджет обновляется каждые 5 секунд
- **Реальное время**: Донаты из DonationAlerts/DonatePay обновляются мгновенно
- **Персистентность**: Все данные сохраняются в базе данных
- **Безопасность**: Валидация всех входных данных
- **Производительность**: Оптимизированные анимации и обновления

## Технические детали

### База данных
- Таблица `donation_goals` - основные настройки цели
- Таблица `goal_donations` - история всех донатов

### Интеграция
- Автоматическое обновление при получении донатов через `processDonation()`
- WebSocket уведомления для обновления в реальном времени
- Совместимость с существующей системой донатов

### Стили
- CSS Grid и Flexbox для адаптивности
- CSS переменные для легкой настройки цветов
- Плавные анимации с cubic-bezier
- Поддержка темной темы
