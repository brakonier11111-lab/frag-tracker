# 🚀 Frag-tracker Server - Бат-файлы для управления

Этот набор бат-файлов обеспечивает полное управление сервером Frag-tracker с автоматической установкой зависимостей, инициализацией базы данных и запуском.

## 📁 Файлы

### 🎯 Основные файлы

- **`server-manager.bat`** - Главное меню управления сервером
- **`start-server-complete.bat`** - Полный запуск (установка + БД + запуск)
- **`start-server-quick.bat`** - Быстрый запуск (только запуск сервера)
- **`stop-server.bat`** - Остановка сервера
- **`update-dependencies.bat`** - Обновление зависимостей
- **`reset-database.bat`** - Сброс базы данных

## 🚀 Быстрый старт

### Первый запуск
1. Запустите **`server-manager.bat`**
2. Выберите пункт **1** (Полный запуск сервера)
3. Дождитесь завершения установки
4. Сервер запустится автоматически

### Последующие запуски
1. Запустите **`server-manager.bat`**
2. Выберите пункт **2** (Быстрый запуск сервера)
3. Сервер запустится сразу

## 📋 Подробное описание

### 🎮 server-manager.bat
**Главное меню управления сервером**

Предоставляет удобное меню для всех операций:
- Полный запуск сервера
- Быстрый запуск сервера
- Остановка сервера
- Обновление зависимостей
- Сброс базы данных
- Проверка состояния
- Открытие веб-интерфейса

### 🚀 start-server-complete.bat
**Полный запуск сервера**

Выполняет полную инициализацию:
1. ✅ Проверка Node.js
2. ✅ Проверка package.json
3. ✅ Установка зависимостей (`npm install`)
4. ✅ Инициализация базы данных
5. ✅ Запуск сервера

**Используйте при:**
- Первом запуске
- После обновления кода
- При проблемах с зависимостями

### ⚡ start-server-quick.bat
**Быстрый запуск сервера**

Быстрый запуск без установки:
1. ✅ Проверка Node.js
2. ✅ Проверка зависимостей
3. ✅ Запуск сервера

**Используйте при:**
- Обычном запуске
- Когда зависимости уже установлены

### 🛑 stop-server.bat
**Остановка сервера**

Останавливает все процессы Node.js:
- Принудительная остановка всех процессов `node.exe`
- Безопасное завершение работы сервера

### 📦 update-dependencies.bat
**Обновление зависимостей**

Обновляет все npm пакеты:
1. ✅ Остановка сервера
2. ✅ Обновление зависимостей (`npm update`)
3. ✅ Готовность к запуску

### 🗑️ reset-database.bat
**Сброс базы данных**

⚠️ **ВНИМАНИЕ! Удаляет ВСЕ данные!**

Полный сброс базы данных:
1. ✅ Подтверждение действия
2. ✅ Остановка сервера
3. ✅ Удаление старой БД
4. ✅ Создание новой БД
5. ✅ Инициализация таблиц

- **Главная:** http://localhost:3000
- **Админ панель:** http://localhost:3000/admin.html
- **Аналитика:** http://localhost:3000/donations-analytics.html
- **Проверка БД:** http://localhost:3000/database-init-check.html

## 🔧 Требования

- **Node.js** версии 14 или выше
- **npm** (устанавливается с Node.js)
- **Windows** с поддержкой бат-файлов

## 🚨 Устранение проблем

### Сервер не запускается
1. Проверьте установку Node.js: `node --version`
2. Запустите полную инициализацию: `start-server-complete.bat`
3. Проверьте логи в консоли

### Ошибки зависимостей
1. Запустите обновление: `update-dependencies.bat`
2. Или полную переустановку: `start-server-complete.bat`

### Проблемы с базой данных
1. Проверьте БД: http://localhost:3000/database-init-check.html
2. При необходимости сбросьте: `reset-database.bat`

### Порт занят
1. Остановите сервер: `stop-server.bat`
2. Проверьте процессы: `tasklist /fi "imagename eq node.exe"`
3. Запустите заново
