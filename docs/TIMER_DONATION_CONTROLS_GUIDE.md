# ⏰ Управление донатами в режиме таймера

## ✅ **Добавленная функциональность:**

### 🔄 **Кнопка "Обнулить"**
- **Расположение:** Под суммой донатов в режиме таймера
- **Функция:** Полное обнуление суммы и количества донатов
- **Подтверждение:** Запрос подтверждения перед выполнением
- **Результат:** Сумма донатов устанавливается в 0₽, все записи удаляются из БД

### 💰 **Кнопка "Стартовая сумма"**
- **Расположение:** Под суммой донатов в режиме таймера
- **Функция:** Установка произвольной стартовой суммы
- **Интерфейс:** Prompt с текущей суммой и полем ввода
- **Валидация:** Автоматическое округление до целых рублей
- **Результат:** Сумма донатов устанавливается в указанное значение

## 🎨 **Дизайн:**

### **Кнопки:**
- **Обнулить:** Красная кнопка с иконкой 🔄
- **Стартовая сумма:** Желтая кнопка с иконкой 💰
- **Стиль:** Неоновые эффекты, hover-анимации
- **Размер:** Компактные кнопки под суммой

### **Уведомления:**
- **Позиция:** Верхний правый угол экрана
- **Анимация:** Плавное появление и исчезновение
- **Цвета:** 
  - Зеленый для успеха
  - Красный для ошибок
  - Синий для информации
- **Автоскрытие:** Через 3 секунды

## 🔧 **Техническая реализация:**

### **API интеграция:**
```javascript
// Обнуление
POST /api/reset-donations-count

// Установка суммы
POST /api/state
{ "total_donated": newAmount }
```

### **Функции:**
- `resetDonationTotal()` - обнуление с подтверждением
- `setStartingAmount()` - установка стартовой суммы
- `showNotification()` - система уведомлений

### **Обновление интерфейса:**
- Мгновенное обновление отображения суммы
- Синхронизация с серверным состоянием
- Обновление всех связанных виджетов

## 📍 **Расположение:**

### **В режиме таймера:**
- **Файл:** `public/mode2-timer.html`
- **Секция:** Статистика таймера
- **Элемент:** Карточка "Всего донатов"
- **Позиция:** Под суммой донатов

## 🎯 **Использование:**

### **Обнуление суммы:**
1. Откройте режим таймера: `http://localhost:3000/mode2-timer.html`
2. Найдите карточку "Всего донатов"
3. Нажмите кнопку "🔄 Обнулить"
4. Подтвердите действие
5. Сумма и количество донатов обнулятся до 0

### **Установка стартовой суммы:**
1. Откройте режим таймера: `http://localhost:3000/mode2-timer.html`
2. Найдите карточку "Всего донатов"
3. Нажмите кнопку "💰 Стартовая сумма"
4. Введите желаемую сумму в prompt
5. Сумма установится в указанное значение
