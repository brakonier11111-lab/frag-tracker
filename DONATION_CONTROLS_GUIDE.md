# 💰 Управление суммой донатов в режиме фрагов

## ✅ **Добавленная функциональность:**

### 🔄 **Кнопка "Обнулить"**
- **Расположение:** Под суммой донатов в режиме фрагов
- **Функция:** Полное обнуление суммы донатов
- **Подтверждение:** Запрос подтверждения перед выполнением
- **Результат:** Сумма донатов устанавливается в 0₽

### 💰 **Кнопка "Стартовая сумма"**
- **Расположение:** Под суммой донатов в режиме фрагов
- **Функция:** Установка произвольной стартовой суммы
- **Интерфейс:** Prompt с текущей суммой и полем ввода
- **Валидация:** Автоматическое округление до целых рублей
- **Результат:** Сумма донатов устанавливается в указанное значение

## 🎨 **Дизайн:**

### **Кнопки:**
- **Обнулить:** Красная кнопка с иконкой 🔄

- **Стиль:** Неоновые эффекты, hover-анимации

## 🔧 **Техническая реализация:**

### **API интеграция:**
```javascript
// Обнуление
POST /api/state
{ "total_donated": 0 }

// Установка суммы
POST /api/state
{ "total_donated": newAmount }
```

### **Функции:**
- `resetDonationTotal()` - обнуление с подтверждением
- `setStartingAmount()` - установка стартовой суммы
- `showNotification()` - система уведомлений

### **Обновление интерфейса:**
- Мгновенное обновление отображения суммы
- Синхронизация с серверным состоянием
- Обновление всех связанных виджетов

## 🎯 **Использование:**

### **Обнуление суммы:**
1. Откройте режим фрагов
2. Найдите карточку "Всего донатов"
3. Нажмите кнопку "🔄 Обнулить"
4. Подтвердите действие
5. Сумма обнулится до 0₽

### **Установка стартовой суммы:**
1. Откройте режим фрагов
2. Найдите карточку "Всего донатов"
3. Нажмите кнопку "💰 Стартовая сумма"
4. Введите желаемую сумму в prompt
5. Сумма установится в указанное значение
