# Redirect URIs для настройки приложения VK Play

## Что вписать в настройках приложения

В разделе **"Список допустимых URL для редиректа после авторизации пользователя"** укажите следующие URL (каждый на отдельной строке):

```
http://localhost:3000/oauth/vkplay/callback
http://localhost:3000/oauth-vkplay-implicit.html
```

## Важно:

1. **Каждый URL на отдельной строке** - не разделяйте запятыми или пробелами
2. **Точное совпадение** - URL должны совпадать символ в символ с теми, что используются в коде
3. **Без слеша в конце** - не добавляйте `/` в конце URL
4. **Без пробелов** - убедитесь, что нет пробелов в начале или конце строки

## Где найти настройки:

1. Откройте панель управления приложением VK Play Live
2. Найдите раздел "Настройки" или "Settings"
3. Найдите поле "Список допустимых URL для редиректа после авторизации пользователя"
4. Вставьте указанные выше URL (каждый на отдельной строке)
5. Сохраните настройки

## Проверка:

После сохранения настройки должны выглядеть так:

```
http://localhost:3000/oauth/vkplay/callback
http://localhost:3000/oauth-vkplay-implicit.html
```

**НЕ так:**
- ❌ `http://localhost:3000/oauth/vkplay/callback, http://localhost:3000/oauth-vkplay-implicit.html` (через запятую)
- ❌ `http://localhost:3000/oauth/vkplay/callback/` (со слешем в конце)
- ❌ ` http://localhost:3000/oauth/vkplay/callback ` (с пробелами)
