## ⚡ Быстрый старт

### Шаг 1: Настройка сбора
1. Откройте вкладку **"Настройки сбора"**
2. Введите название и целевую сумму
3. Нажмите **"Сохранить настройки"**

### Шаг 2: Выбор стиля полоски
1. Перейдите во вкладку **"Темы полоски"**
2. Выберите понравившуюся тему
3. Готово! Тема применена автоматически

### Шаг 3: Получение ссылки для OBS
1. Откройте вкладку **"Ссылки для OBS"**
2. Выберите нужную ссылку
3. Нажмите **"Копировать"**
4. Добавьте в OBS как Browser Source

## 🔧 Настройка OBS Browser Source

### Пошаговая инструкция:

1. **Откройте OBS Studio**

2. **Добавьте Browser Source:**
   - Источники → Добавить → Browser

3. **Настройте источник:**
   - **URL:** Вставьте скопированную ссылку
   - **Ширина/Высота:** Установите рекомендуемые размеры
   - ☑️ **Shutdown source when not visible** (для экономии ресурсов)
   - ☐ Убедитесь, что галочка "Local file" снята

4. **Настройте позицию:**
   - Перетащите виджет в нужное место
   - Измените размер при необходимости

5. **Готово!** ✅

## 💾 Сохранение и восстановление

### Сохранение настроек:
- Настройки сбора сохраняются на сервере
- Настройки полоски сохраняются в браузере
- Используйте кнопку **"Сохранить настройки"** в каждой вкладке

### Восстановление:
- При открытии страницы все настройки загружаются автоматически
- Настройки синхронизируются между вкладками

### Экспорт конфигурации:
1. Настройте все параметры
2. Скопируйте ссылку из вкладки **"Ссылки для OBS"**
3. Сохраните ссылку в текстовый файл
4. При необходимости используйте эту ссылку

## 🐛 Решение проблем

### Виджет не обновляется
- Проверьте, что сервер запущен
- Обновите страницу (F5)
- Проверьте консоль браузера на ошибки

### Изменения не сохраняются
- Убедитесь, что нажали кнопку "Сохранить настройки"
- Проверьте, что cookies и localStorage разрешены
- Попробуйте другой браузер

### Превью не отображается
- Проверьте настройки блокировщика рекламы
- Разрешите iframe для localhost
- Обновите страницу

### Виджет не отображается в OBS
- Проверьте правильность URL
- Убедитесь, что сервер запущен
- Проверьте размеры источника
- Обновите источник в OBS (Ctrl+R)
