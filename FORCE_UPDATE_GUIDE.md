# 🔄 Принудительное обновление виджета OBS

## ✅ Что изменено в виджете:

### 🎨 **Кардинальные изменения дизайна:**
- **Шрифт Inter**: принудительно применен с !important
- **Вес шрифта**: увеличен до 800 (очень жирный)
- **Размер**: увеличен до 1.4rem
- **Межбуквенный интервал**: увеличен до 1px
- **Заглавные буквы**: text-transform: uppercase
- **Усиленное свечение**: более яркие тени
- **Версия стилей**: v2.0 для принудительного обновления

## 🚀 **Как принудительно обновить:**

### 1. **В браузере:**
- **Ctrl + F5** - принудительное обновление
- **Ctrl + Shift + R** - полная перезагрузка
- **F12** → Network → Disable cache → обновить

### 2. **В OBS:**
- **Перезапустить** Browser Source
- **Обновить URL**: `http://localhost:3000/widget/donation-goal?v=2.0`
- **Очистить кэш** OBS (Settings → Advanced → Browser Cache)

### 3. **Если все еще не работает:**
- **Перезапустить сервер**: остановить и запустить `node server.js`
- **Проверить консоль**: F12 → Console на ошибки
- **Проверить Network**: загружается ли шрифт Inter

## 🎯 **Ожидаемый результат:**
- **ЗАГЛАВНЫЕ БУКВЫ** в названии сбора
- **ОЧЕНЬ ЖИРНЫЙ** шрифт Inter
- **УСИЛЕННОЕ СВЕЧЕНИЕ** с анимацией
- **УВЕЛИЧЕННЫЙ РАЗМЕР** текста
- **СОВРЕМЕННЫЙ ДИЗАЙН** вместо стандартного

## 🔍 **Как проверить изменения:**
1. Откройте `http://localhost:3000/widget/donation-goal` в браузере
2. Название должно быть **ЗАГЛАВНЫМИ БУКВАМИ**
3. Шрифт должен быть **очень жирным**
4. Должно быть **яркое свечение**

---
*Если изменения все еще не видны, попробуйте добавить `?v=2.0` к URL в OBS! 🔄*
