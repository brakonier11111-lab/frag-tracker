## 🎯 Обзор

Новая система полосок донатов предоставляет гибкую кастомизацию с поддержкой различных ориентаций, стилей и визуальных эффектов.

### Основные возможности:
- ✅ 7 вариантов ориентации (горизонтальная, вертикальная, круговая, 4 угловых)
- ✅ 4 визуальных стиля (стандартный, минималистичный, неоновый, радуга)
- ✅ Полная настройка цветов и размеров
- ✅ Управление видимостью элементов
- ✅ Настраиваемая скорость анимации
- ✅ Выбор шрифтов
- ✅ Сохранение/загрузка настроек

## ⚙️ Настройка через интерфейс

### Открытие панели настроек
1. Откройте `donation-bar-customizable.html`
2. Нажмите кнопку **⚙️ Настройки** в левом верхнем углу
3. Настройте параметры в реальном времени

### Основные настройки

#### 🎯 Ориентация
Выберите один из 7 вариантов расположения

#### 🎨 Цвета
- **Цвет фона полоски** - Основной фон прогресс-бара
- **Цвет заполнения** - Цвет заполненной части
- **Цвет границы** - Цвет обводки
- **Цвет свечения** - Цвет эффекта свечения

#### 📏 Размеры
- **Высота/Толщина** - От 20px до 100px
- **Скругление углов** - От 0px (прямоугольник) до 50px (круглые концы)

#### ⚡ Анимация
- **Скорость анимации** - От 0.1x (медленно) до 3x (быстро)

#### 🔤 Шрифт
Выберите из доступных шрифтов:
- Inter (современный, универсальный)
- Roboto (чистый, читабельный)
- Montserrat (элегантный)
- Arial (классический)
- Courier New (моноширинный)

#### 👁️ Отображение элементов
- ☑️ **Показать название** - Заголовок сбора
- ☑️ **Показать сумму внутри** - Текущая сумма в центре полоски
- ☑️ **Показать начало/конец** - Стартовая и целевая суммы
- ☐ **Показать процент** - Процент достижения цели
- ☑️ **Показать частицы** - Анимированные частицы

### Сохранение настроек
1. Настройте все параметры по желанию
2. Нажмите кнопку **Сохранить настройки**
3. Настройки сохраняются в браузере (localStorage)
4. При следующем открытии настройки загрузятся автоматически

### Сброс настроек
Нажмите **Сбросить настройки** для возврата к стандартным значениям

## 🔗 Настройка через URL для OBS

### Базовая структура URL
```
http://localhost:3000/donation-bar-obs.html?параметр1=значение1&параметр2=значение2
```

### Доступные параметры

| Параметр | Описание | Возможные значения | По умолчанию |
|----------|----------|-------------------|--------------|
| `orientation` | Ориентация | `horizontal`, `vertical`, `circular`, `corner-tl`, `corner-tr`, `corner-bl`, `corner-br` | `horizontal` |
| `style` | Визуальный стиль | `default`, `minimal`, `neon`, `rainbow` | `default` |
| `bgColor` | Цвет фона | HEX (без #) | `000000` |
| `fillColor` | Цвет заполнения | HEX (без #) | `4a5568` |
| `borderColor` | Цвет границы | HEX (без #) | `4a5568` |
| `glowColor` | Цвет свечения | HEX (без #) | `667eea` |
| `height` | Высота/толщина | Число (px) | `35` |
| `radius` | Скругление | Число (px) | `30` |
| `animSpeed` | Скорость анимации | Число (0.1-3) | `1` |
| `font` | Шрифт | `Inter`, `Roboto`, `Montserrat`, `Arial`, `Courier New` | `Inter` |
| `showTitle` | Показать название | `true`, `false` | `true` |
| `showAmount` | Показать сумму | `true`, `false` | `true` |
| `showStartEnd` | Начало/конец | `true`, `false` | `true` |
| `showPercentage` | Показать процент | `true`, `false` | `false` |
| `showParticles` | Показать частицы | `true`, `false` | `true` |

## 🎓 Советы и рекомендации

### Для стримеров

#### 1. Выбор позиции
- **Горизонтальная** - Верх или низ экрана
- **Вертикальная** - Боковые панели
- **Круговая** - Углы экрана
- **Угловые** - Максимально компактно в углах

#### 2. Цветовая схема
- Используйте контрастные цвета для лучшей видимости
- Учитывайте цветовую гамму вашего оверлея
- Яркие цвета для привлечения внимания
- Темные для ненавязчивого отображения

#### 3. Размеры для OBS
**Горизонтальная:**
- Маленькая: 600x120px
- Средняя: 800x150px
- Большая: 1000x180px

**Вертикальная:**
- Маленькая: 120x400px
- Средняя: 150x600px
- Большая: 180x800px

**Круговая:**
- Маленькая: 250x300px
- Средняя: 350x400px
- Большая: 450x500px

#### 4. Производительность
- Минималистичный стиль потребляет меньше ресурсов
- Отключите частицы для экономии GPU
- Уменьшите скорость анимации для стабильности

### Для дизайнеров

#### 1. Сочетание цветов
**Классические:**
- Синий (#667eea) + Фиолетовый (#764ba2)
- Зеленый (#4CAF50) + Темно-зеленый (#2E7D32)
- Красный (#ff6b6b) + Оранжевый (#ee5a24)

**Неоновые:**
- Голубой (#00ffff) + Фиолетовый (#ff00ff)
- Зеленый (#00ff00) + Желтый (#ffff00)
- Розовый (#ff69b4) + Фиолетовый (#9400d3)

**Минималистичные:**
- Белый (#ffffff) + Серый (#808080)
- Черный (#000000) + Серый (#4a5568)

#### 2. Типографика
- **Inter** - Лучший выбор для универсальности
- **Roboto** - Для чистого современного вида
- **Montserrat** - Для элегантного стиля
- **Orbitron** - Для киберпанк/технологичного стиля

#### 3. Анимация
- **0.5x** - Медленная, плавная
- **1x** - Стандартная, сбалансированная
- **2x** - Быстрая, энергичная
- **3x** - Очень быстрая, для динамичных стримов

### Комбинирование с другими виджетами

#### Сочетание с alert'ами
Используйте похожую цветовую схему для единого стиля:
```
// Полоска
?fillColor=667eea&glowColor=667eea

// Alert должен использовать те же цвета
```

#### Адаптация под игры
- **CS:GO/Valorant** - Красные и оранжевые тона
- **League of Legends** - Синие и золотые
- **Minecraft** - Зеленые и коричневые
- **Cyberpunk** - Неоновые розовые и голубые

## 🐛 Решение проблем

### Виджет не отображается в OBS
1. Проверьте правильность URL
2. Убедитесь, что сервер запущен
3. Проверьте размеры источника браузера
4. Попробуйте обновить источник (Ctrl+R)

### Анимация тормозит
1. Уменьшите скорость анимации (`animSpeed=0.5`)
2. Используйте минималистичный стиль
3. Отключите частицы (`showParticles=false`)

### Цвета отображаются некорректно
1. Убедитесь, что HEX-код указан без символа #
2. Проверьте корректность кода (6 символов)
3. Используйте только допустимые символы (0-9, A-F)
