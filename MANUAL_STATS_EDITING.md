# ✏️ Ручное редактирование статистики

## 🎯 Описание
Система для ручного редактирования статистики боев и фрагов. Позволяет изменить количество боев, фрагов и боев без фрагов для продолжения отсчета с новых значений.

## 🚀 Способы редактирования

### 1. Через админ-панель (рекомендуется)
1. Откройте http://localhost:3000/admin
2. Нажмите "✏️ РЕДАКТИРОВАТЬ СТАТИСТИКУ"
3. Система покажет текущие значения
4. Введите новые значения:
   - Количество боев
   - Количество фрагов
   - Количество боев без фрагов
5. Подтвердите редактирование

### 2. Через скрипт
```bash
# Отредактируйте файл edit-stats-manually.js
# Измените параметры:
#   newTotalBattles = 25
#   newTotalFrags = 50
#   newBattlesWithoutFrags = 5

# Запустите скрипт
edit-stats-manually.bat
```

### 3. Через API
```bash
curl -X POST http://localhost:3000/api/edit-stats-manually \
  -H "Content-Type: application/json" \
  -d '{
    "totalBattles": 25,
    "totalFrags": 50,
    "battlesWithoutFrags": 5
  }'
```

## 📊 Примеры редактирования

### Пример 1: Увеличение статистики
```
Было: 19 боев, 37 фрагов, 4 боя без фрагов
Стало: 25 боев, 50 фрагов, 5 боев без фрагов

Результат:
• 20 боев с фрагами:
  - 1-й бой: 2 + 10 = 12 фрагов
  - 19 боев: по 2 фрага
• 5 боев: по 0 фрагов
```

### Пример 2: Уменьшение статистики
```
Было: 30 боев, 60 фрагов, 10 боев без фрагов
Стало: 20 боев, 40 фрагов, 5 боев без фрагов

Результат:
• 15 боев с фрагами:
  - 1-й бой: 2 + 10 = 12 фрагов
  - 14 боев: по 2 фрага
• 5 боев: по 0 фрагов
```

### Пример 3: Изменение только фрагов
```
Было: 20 боев, 30 фрагов, 5 боев без фрагов
Стало: 20 боев, 45 фрагов, 5 боев без фрагов

Результат:
• 15 боев с фрагами:
  - 1-й бой: 3 + 0 = 3 фрага
  - 14 боев: по 3 фрага
• 5 боев: по 0 фрагов
```

## 🔧 Как работает редактирование

### 1. Получение текущих значений
- Система получает текущую статистику из базы данных
- Показывает количество боев, фрагов и боев без фрагов
- Позволяет редактировать каждое значение отдельно

### 2. Расчет новых значений
```
Боев с фрагами = Всего боев - Боев без фрагов
Фрагов за бой = Всего фрагов ÷ Боев с фрагами (целая часть)
Остаток фрагов = Всего фрагов - (Фрагов за бой × Боев с фрагами)
```

### 3. Распределение фрагов
- Первый бой получает остаток фрагов
- Остальные бои получают равное количество фрагов
- Бои без фрагов получают 0 фрагов

### 4. Обновление данных
- Очищается существующая статистика
- Создаются новые записи боев
- Обновляется состояние Lesta Games
- Система готова к работе с новыми значениями

## ⚠️ Важные моменты

### 1. Перезапись данных
- **ВНИМАНИЕ**: Редактирование полностью перезаписывает статистику
- Предыдущие данные будут потеряны
- Сделайте резервную копию перед редактированием

### 2. Валидация данных
- Количество боев без фрагов не может быть больше общего количества боев
- Количество фрагов не может быть отрицательным
- Система проверяет корректность введенных данных

### 3. Синхронизация
- После редактирования обновляется состояние Lesta Games
- Система будет отслеживать изменения от новых значений
- Автообновление продолжит работать корректно
