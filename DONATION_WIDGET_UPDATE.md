# 🎯 Обновления виджета сбора донатов

## ✨ Новые изменения:

### 🎨 **Анимация стала еще медленнее:**
- **Переход полоски**: 4s (было 2.5s)
- **Анимация градиента**: 12s (было 6s) 
- **Анимация свечения**: 6s (было 4s)
- **Цвета**: темные оттенки серого для спокойного вида

### 📱 **Виджет для OBS улучшен:**
- **Название сбора**: отображается над полоской с красивым свечением
- **Процент внутри полоски**: размещен посередине с отчетливой видимостью
- **Темная анимация**: спокойные переходы цветов
- **Прозрачный фон**: идеально для OBS

### 🔧 **Исправлены ошибки:**
- **Сохранение настроек**: работает корректно
- **Добавление донатов**: без ошибок
- **Автосоздание таблиц**: при запуске сервера

## 🚀 **Как использовать:**

### 1. **Запуск сервера:**
```bash
node server.js
```

### 2. **Управление виджетом:**
- **Страница управления**: `http://localhost:3000/donation-goal-widget.html`
- **Добавление донатов**: вручную через форму
- **Настройка параметров**: название, сумма, срок

### 3. **Виджет для OBS:**
- **Ссылка**: `http://localhost:3000/widget/donation-goal`
- **Размер**: настраивается в OBS
- **Фон**: прозрачный, только полоска прогресса

## 🎯 **Особенности виджета OBS:**
- **Название сбора** сверху с анимацией свечения
- **Полоска прогресса** с медленной темной анимацией
- **Процент** внутри полоски посередине
- **Суммы** снизу (текущая и целевая)
- **Плавающие частицы** вокруг полоски

## 📊 **Функции управления:**
- ✅ Добавление ручных донатов
- ✅ Изменение названия сбора
- ✅ Настройка целевой суммы
- ✅ Установка срока сбора
- ✅ Обнуление прогресса
- ✅ История донатов
- ✅ Экспорт данных

## 🎨 **Дизайн:**
- **Темные цвета**: спокойные серые оттенки
- **Медленная анимация**: плавные переходы
- **Свечение**: приглушенное и элегантное
- **Современный вид**: как у топовых стримеров
