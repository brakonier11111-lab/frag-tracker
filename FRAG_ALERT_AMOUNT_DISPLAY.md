# 🎯 Отображение суммы до фрага в алертах

## ✨ Новая функциональность
Теперь в алертах фрагов отображается сумма до фрага, если донат не закрывает полоску до фрага.

## 🔄 Логика работы

### Если донат закрывает полоску до фрага:
- **Отображается**: `+2 фрага` (количество фрагов)
- **Тип**: `fragDisplayType: 'frags'`

### Если донат НЕ закрывает полоску до фрага:
- **Отображается**: `Добавил до +1 фрага` (фиксированная надпись)
- **Тип**: `fragDisplayType: 'amount'`

## 🚀 Реализация

### 1. Серверная логика (server.js)
```javascript
// Логика отображения в алерте
if (fragUnitsEarned > 0) {
    // Если донат закрывает полоску до фрага - показываем количество фрагов
    alertData.fragsEarned = fragUnitsEarned;
    alertData.fragUnitName = state.frag_name;
    alertData.fragDisplayType = 'frags'; // Тип отображения: фраги
} else {
    // Если донат не закрывает полоску - показываем "Добавил до +1 фрага"
    alertData.fragsEarned = 0;
    alertData.fragUnitName = state.frag_name;
    alertData.fragDisplayType = 'amount'; // Тип отображения: сумма
}
```

### 2. Клиентская логика (alert-mode1.html)
```javascript
const frags = (donation.fragsEarned != null ? donation.fragsEarned : 0) || 0;
const fragName = donation.fragUnitName || 'фрагов';
const fragDisplayType = donation.fragDisplayType || 'frags';

if (frags > 0) {
    // Если донат закрывает полоску до фрага - показываем количество фрагов
    document.getElementById('fragsEarned').textContent = `+${frags} ${fragName}`;
    document.getElementById('fragsEarned').style.display = 'block';
} else if (fragDisplayType === 'amount') {
    // Если донат не закрывает полоску - показываем "Добавил до +1 фрага"
    document.getElementById('fragsEarned').textContent = `Добавил до +1 ${fragName}`;
    document.getElementById('fragsEarned').style.display = 'block';
} else {
    document.getElementById('fragsEarned').style.display = 'none';
}
```

## 📊 Примеры работы

### Пример 1: Донат закрывает полоску

Донат: 200₽
Стоимость фрага: 100₽
Текущий баланс: 50₽

Расчет: (50 + 200) / 100 = 2 фрага, остаток: 50₽
Отображение: "+2 фрага"
```

### Пример 2: Донат не закрывает полоску

Стоимость фрага: 100₽

Расчет: (50 + 30) / 100 = 0 фрагов, остаток: 80₽
Отображение: "Добавил до +1 фрага"
```
