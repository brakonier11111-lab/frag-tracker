# 🔧 Исправление проблемы с DonationAlerts

## ❌ Проблема
Донаты с DonationAlerts не отображаются в программе, несмотря на наличие API ключей и ссылок на виджет.

## 🔍 Диагностика

### 1. Откройте диагностику опроса
**URL:** http://localhost:3000/admin

**Действие:** Нажмите кнопку **"🔍 ДИАГНОСТИКА ОПРОСА ДОНАТОВ"**

Эта диагностика покажет:
- ✅ Есть ли токен доступа DonationAlerts
- ✅ Работает ли API получения донатов
- ✅ Активен ли опрос донатов
- ✅ Сколько донатов получено

### 2. Проверьте статус системы
**URL:** http://localhost:3000/api/status

Проверьте:
- `donationAlerts.hasToken` - должен быть `true`
- `polling.isPollingInProgress` - статус опроса
- `polling.hasPollingInterval` - активен ли интервал

### 3. Тестовая страница
**URL:** http://localhost:3000/donation-alerts-test.html

Нажмите все кнопки по порядку:
1. **Проверить статус** - общий статус системы
2. **Тест OAuth** - проверка авторизации
3. **Тест API** - проверка получения донатов
4. **Тест виджета** - проверка виджета
5. **Диагностика** - подробная диагностика

## 🚀 Исправления

### Исправление 1: Добавлен вызов addFragStats
**Проблема:** Донаты обрабатывались, но фраги не добавлялись в статистику.

**Решение:** Добавлен вызов `addFragStats` в функцию `processDonation`:

```javascript
if (fragUnitsEarned > 0) {
    updatedState.frags_needed = (state.frags_needed || 0) + fragUnitsEarned;
    
    // Добавляем фраги в статистику
    const battleTime = new Date().toISOString();
    addFragStats(battleTime, fragUnitsEarned);
    console.log(`📊 Добавлено ${fragUnitsEarned} фрагов в статистику от доната ${username}`);
}
```

- ✅ Диагностика в `checkForNewDonations`

- ✅ Кнопка диагностики в админ-панели

### Исправление 3: Улучшена обработка донатов
**Добавлено:**
- ✅ Логирование источника доната (REALTIME/POLLING)

## 🔧 Возможные проблемы и решения

### Проблема 1: Нет токена доступа
**Симптомы:**
- Диагностика показывает "Токен: НЕТ"
- API тест возвращает ошибку "Access token не настроен"

**Решение:**
1. Откройте http://localhost:3000/admin
2. Нажмите "🔗 АВТОРИЗАЦИЯ DONATIONALERTS"
3. Выполните OAuth авторизацию
4. Скопируйте код из URL после перенаправления
5. Введите код в админ-панели

### Проблема 2: Токен устарел
**Симптомы:**
- API возвращает ошибку 401 Unauthorized
- В логах: "Токен устарел, требуется повторная авторизация"

**Решение:**
1. Выполните повторную OAuth авторизацию
2. Получите новый токен доступа

### Проблема 3: API не возвращает донаты
**Симптомы:**
- Токен есть, но донатов нет
- Диагностика показывает "0 донатов"

**Решение:**
1. Проверьте права доступа в DonationAlerts
2. Убедитесь, что используете правильный scope: `oauth-donation-index`
3. Проверьте, есть ли донаты в вашем аккаунте DonationAlerts

### Проблема 4: Опрос не запущен
**Симптомы:**
- Диагностика показывает "Активен: НЕТ"
- Донаты не обрабатываются автоматически

**Решение:**
1. Перезапустите сервер
2. Проверьте, что есть хотя бы один токен (DonationAlerts или DonatePay)
3. Опрос запустится автоматически
