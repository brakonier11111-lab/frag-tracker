# Исправление ошибки invalid_scope в VK Play OAuth

## Проблема
При авторизации в VK Play возникает ошибка `invalid_scope`, даже когда параметр `scope` не указан в URL авторизации.

## Возможные причины

1. **Настройки приложения в VK Play** - Приложение может быть настроено с неверными scope в личном кабинете VK Play
2. **Автоматическое добавление scope** - VK Play может автоматически добавлять scope на основе настроек приложения
3. **Требование scope** - VK Play может требовать указания scope, но мы их не указываем
4. **Не выбраны нужные события** - В настройках приложения может быть выбрано только 1 событие, но нужно больше

## Решения

### Решение 1: Проверьте настройки приложения в VK Play

1. Откройте личный кабинет VK Play для разработчиков
2. Найдите ваше приложение (ID: as34hljtetqhik1e)
3. Перейдите в раздел "Настройки" или "Settings"
4. Найдите раздел "Разрешения" или "Permissions" / "Scope"
5. Убедитесь, что все необходимые разрешения включены:
   - Управление наградами за баллы канала
   - Запросы наград за баллы канала
   - Управление ролями
   - Отправка сообщений в чат
6. Проверьте раздел "События" или "Events":
   - Убедитесь, что выбраны все необходимые события
   - Для работы с наградами нужно событие активации награды
   - Для работы с чатом нужно событие сообщения в чате
7. Если есть неверные или несуществующие scope, удалите их

### Решение 2: Попробуйте указать scope в URL (если поддерживается)

Раскомментируйте один из вариантов в `server.js`:

```javascript
// Вариант 1: Через запятую
const scopes = 'channel:points,channel:points:rewards,channel:points:rewards:demands,channel:roles,channel:chat:write';
params.scope = scopes;

// Вариант 2: Через пробел
const scopes = 'channel:points channel:points:rewards channel:points:rewards:demands channel:roles channel:chat:write';
params.scope = scopes;
```

### Решение 3: Проверьте документацию VK Play Live API

Откройте документацию по адресу:
- https://dev.live.vkvideo.ru/docs/main/authorization
- https://dev.live.vkvideo.ru/docs/method/token

Найдите точные названия scope и формат их указания.

### Решение 4: Создайте новое приложение в VK Play

Если проблема не решается, попробуйте:
1. Создать новое приложение в VK Play
2. Настроить его с правильными разрешениями
3. Использовать новые `client_id` и `client_secret`

## Текущее состояние

В коде scope убраны из параметров авторизации, чтобы избежать ошибки `invalid_scope`. 

**Важно:** VK Play Live API может не поддерживать указание scope в URL авторизации. Разрешения должны быть настроены в панели управления приложением VK Play, а не передаваться в URL.

Если ошибка все еще возникает, это означает, что проблема в настройках приложения в VK Play, а не в коде.
