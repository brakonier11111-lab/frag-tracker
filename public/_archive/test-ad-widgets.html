📋 Инструкции по использованию

  • Полноценный виджет - для крупных сцен, показывает полную информацию о режимах
  • Компактный виджет - для средних сцен, баланс между информативностью и размером
  • Полоска режимов - для маленьких сцен или как дополнительный элемент
  • Все виджеты автоматически переключают режимы каждые 2.5-4 секунды
  • При наведении мыши ротация останавливается
  • Клик по индикаторам переключает режим вручную
  • Все виджеты имеют прозрачный фон для OBS

🎮 Для OBS:

  • Добавьте источник "Browser Source"
  • Вставьте URL нужного виджета
  • Установите размер в соответствии с рекомендациями
  • Включите "Shutdown source when not visible" для экономии ресурсов
