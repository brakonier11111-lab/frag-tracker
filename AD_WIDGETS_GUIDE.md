# 🎯 Рекламные виджеты для стрима

## 📱 **Созданные виджеты:**

### 1. **Полноценный виджет** (`widget-modes-ad.html`)
- **Размер:** 300x200px
- **Особенности:** 
  - Полное описание каждого режима
  - Большие иконки и анимации
  - Индикаторы режимов
  - Призыв к действию
- **Для:** Крупных сцен, основного показа

### 2. **Компактный виджет** (`widget-modes-compact.html`)
- **Размер:** 300x150px
- **Особенности:**
  - Минималистичный дизайн
  - Только иконка и название режима
  - Точки-индикаторы
  - Краткий призыв
- **Для:** Средних сцен, дополнительной информации

### 3. **Полоска режимов** (`widget-modes-strip.html`)
- **Размер:** 300x80px
- **Особенности:**
  - Горизонтальная компоновка
  - Минимальный текст
  - Только суть
- **Для:** Маленьких сцен, как дополнительный элемент

## 🎮 **Режимы таймера:**

| Режим | Иконка | Цвет | Описание |
|-------|--------|------|----------|
| **Обычный** | ⏰ | Синий | Классический таймер |
| **Скидка** | 🎁 | Желтый | Временные скидки |
| **Замедление** | 🐢 | Фиолетовый | Замедление времени |
| **Перегрев** | 🔥 | Красный | Система температуры |

## ⚙️ **Функциональность:**

### **Автоматическая ротация:**
- **Полноценный:** 4 секунды
- **Компактный:** 3 секунды  
- **Полоска:** 2.5 секунды

### **Интерактивность:**
- ✅ **Клик по индикаторам** - переключение режима
- ✅ **Hover** - пауза ротации
- ✅ **Автопауза** при скрытии страницы

### **Анимации:**
- 🔄 **Смена режимов** - плавные переходы
- ✨ **Свечение** - неоновые эффекты
- 🌟 **Shimmer** - фоновые блики
- 💫 **Pulse** - пульсация иконок

## 🎬 **Для OBS:**

### **Настройка Browser Source:**
1. Добавить источник "Browser Source"
2. Вставить URL: `http://localhost:3000/widget-modes-[тип].html`
3. Установить размер в соответствии с рекомендациями
4. Включить "Shutdown source when not visible"

### **Рекомендуемые размеры:**
- **Полноценный:** 300x200px
- **Компактный:** 300x150px
- **Полоска:** 300x80px
