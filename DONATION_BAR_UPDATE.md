# 🎨 Обновление: Кастомизация полосок донатов

## 🎉 Что нового?

Добавлена полная система кастомизации полосок донатов с множеством вариантов отображения и стилей!

---

## ✨ Новые возможности

### 🔄 7 вариантов ориентации
- **Горизонтальная** - классическая полоска прогресса
- **Вертикальная** - боковая панель с прогрессом
- **Круговая** - стильная круговая диаграмма
- **Угловые (4 варианта)** - компактные индикаторы в углах экрана

### 🎨 4 визуальных стиля
- **Стандартный** - с градиентами и свечением
- **Минималистичный** - чистый плоский дизайн
- **Неоновый** - яркие эффекты свечения
- **Радуга** - анимированный радужный градиент

### ⚙️ Гибкая настройка
- Выбор любых цветов для фона, заполнения, границы и свечения
- Настройка высоты/толщины полоски (20-100px)
- Регулировка скругления углов (0-50px)
- Управление скоростью анимации (0.1x - 3x)
- Выбор шрифта (Inter, Roboto, Montserrat, Arial, Courier New)

### 👁️ Управление видимостью
- Показать/скрыть название сбора
- Показать/скрыть текущую сумму
- Показать/скрыть начальную и целевую суммы
- Показать/скрыть процент выполнения
- Показать/скрыть анимированные частицы

## 🚀 Как начать использовать

### Способ 1: Готовая тема (самый быстрый)
1. Откройте [donation-bar-themes.html](http://localhost:3000/donation-bar-themes.html)
2. Выберите любую тему
3. Нажмите "Получить ссылку"
4. Добавьте в OBS как Browser Source

### Способ 2: Визуальный редактор
1. Откройте [donation-bar-customizable.html](http://localhost:3000/donation-bar-customizable.html)
2. Нажмите кнопку ⚙️ **Настройки**
3. Настройте все параметры визуально
4. Сохраните настройки

### Способ 3: URL параметры (для продвинутых)
1. Создайте URL с нужными параметрами
2. Пример: `http://localhost:3000/donation-bar-obs.html?orientation=circular&fillColor=667eea&style=neon`
3. Используйте в OBS

## 🎨 Примеры URL параметров

```
http://localhost:3000/donation-bar-obs.html?orientation=horizontal&style=minimal&fillColor=4CAF50&showParticles=false
```

### Неоновая круговая диаграмма
```
http://localhost:3000/donation-bar-obs.html?orientation=circular&style=neon&fillColor=ff00ff&showPercentage=true
```

```
http://localhost:3000/donation-bar-obs.html?style=rainbow&animSpeed=2&height=50
```

### Компактный угловой индикатор
```
http://localhost:3000/donation-bar-obs.html?orientation=corner-tr&style=minimal&showTitle=false&showStartEnd=false
```

## 💡 Полезные советы

### Для привлечения внимания
- Используйте яркие цвета (красный, золотой, неоновые)
- Включите стиль `rainbow` или `neon`
- Увеличьте скорость анимации (`animSpeed=2`)

### Для ненавязчивого отображения
- Используйте темные или пастельные цвета
- Выберите стиль `minimal`
- Уменьшите скорость анимации (`animSpeed=0.5`)
- Отключите частицы (`showParticles=false`)

### Для экономии ресурсов ПК
- Стиль `minimal`
- Без частиц (`showParticles=false`)
- Низкая скорость анимации (`animSpeed=0.5`)

### Для разных игр
- **Экшн-игры** - красные/оранжевые тона, быстрая анимация
- **Расслабляющие игры** - синие/зеленые тона, медленная анимация
- **Хоррор** - темные тона, минимум эффектов
- **Киберпанк** - неоновые розовые/голубые, яркое свечение

## 🐛 Известные ограничения

- Для изменения названия сбора и целевой суммы используйте основной виджет [donation-goal-widget.html](http://localhost:3000/donation-goal-widget.html)
- Настройки в визуальном редакторе сохраняются локально в браузере
- URL параметры имеют приоритет над сохраненными настройками
- Некоторые браузеры могут не поддерживать функцию `color-mix()` (используется в градиентах)
