# 💰 Система управления донатами - Краткая справка

## 🚀 Быстрый доступ

### Главная страница управления
```
http://localhost:3000/donation-management.html
```

**Через меню:** Нажмите **💰 Управление донатами**

---

## 📱 Структура системы

### 🎯 6 вкладок в одном месте:

| Вкладка | Назначение | Что можно делать |
|---------|-----------|------------------|
| **🎯 Настройки сбора** | Управление целью | Название, сумма, даты, ручные донаты |
| **📊 Полный виджет** | Просмотр виджета | Полнофункциональный виджет с панелью управления |
| **🎨 Темы полоски** | Готовые стили | 8 готовых тем одним кликом |
| **⚙️ Кастомизация** | Тонкая настройка | Цвета, размеры, эффекты + превью |
| **🔗 Ссылки для OBS** | Browser Source | 3 готовые ссылки для копирования |
| **📜 История** | Все донаты | Полный список с деталями |

---

## ⚡ Быстрый старт за 3 шага

### Шаг 1: Настройте сбор
1. Откройте вкладку **"Настройки сбора"**
2. Заполните название и целевую сумму
3. Сохраните

### Шаг 2: Выберите стиль
1. Вкладка **"Темы полоски"**
2. Кликните на любую тему

### Шаг 3: Добавьте в OBS
1. Вкладка **"Ссылки для OBS"**
2. Скопируйте ссылку
3. Browser Source в OBS

**Готово! ✅**

## 🐛 Частые вопросы

### Как изменить целевую сумму?
Вкладка **"Настройки сбора"** → Изменить → Сохранить

### Как применить тему?
Вкладка **"Темы полоски"** → Клик по теме

### Как создать свой стиль?
Вкладка **"Кастомизация"** → Настроить → Сохранить

### Как получить ссылку для OBS?
Вкладка **"Ссылки для OBS"** → Копировать

### Как добавить донат вручную?
Вкладка **"Настройки сбора"** → Ручной донат → Добавить

### Где посмотреть историю?
Вкладка **"История"**
